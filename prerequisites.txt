Prerequisites: sox, aplay and rtl_udp(built here), cmake, build-essential or gcc (usual build tools)
gnuradio rtl-sdr socat

One or Two RTL2838 Dongles, 1 for Tracking, 2 for Tracking and Tuning

There may be other build requirements, please submit issue if cannot build or
determine build requirements that are missing here and which platform. Thanks.
Raspberry Pi: libusb-1.0-0-dev
Build-Essential, Sox: Just run (on debian/ubuntu/pi, etc)

sudo apt update
sudo apt install sox cmake build-essential socat libusb-1.0-0-dev libncurses5 libncurses5-dev gnuradio rtl-sdr git

chmod +x build.sh rebuild.sh start-control-gr.sh start-control-rtlfm.sh start-lcn-gr.sh start-lcn-rtludp.sh start-pyedacs-gr.sh sdrpp-gqrx-udp-signal-analyzer.sh

to give these shell scripts execute permission, then run

./build.sh

to build the edacs-fm, edacs-fm, and rtl_udp applications

if you wish to build only edacs-fm or edacs-lcn, or both, then you
simply only need to run

gcc -o edacs-fm edacs-fm.c $(ncursesw5-config --cflags --libs)
gcc -o edacs-lcn edacs-lcn.c
